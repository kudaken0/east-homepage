【除隊】

1.過度な煽り・暴言・下ネタ

過度な煽り ... ゲーム上での死体撃ち、相手が不快になる言動など過度な煽りと判断した場合除隊となります

暴言 ... 相手なすっっっっっっっごい嫌がるような言葉

下ネタ ... 言わなくてもわかりますね^^

2. クラ活に3回連続で来ない

定期的にやるクラ活に3回来なかった場合除隊になります。

3. フォートナイトの規約違反

チーミング、チート、バグや誤作動の悪用その他不正行為

詳しくはこちら

4. 犯罪に問われる行為

児童ポルノ、万引きその他犯罪に問われる行為が発覚した場合直ちに除隊します。

5. クランの印象を下げる行為

配信者への荒らし、ゴースティング、不適切なツイート・ストーリーなど

厳重注意

煽り・暴言・下ネタ

煽り ... 除隊の1番に書いた過度な煽りと判断しなかった場合でもさすがにやりすぎると判断した場合厳重注意

暴言 ... さっき書いた煽りとほぼ同じ内容()

下ネタ ... 幼稚園児じゃないんだからわかりますね^^

2. スパムなどの荒らし行為

例: 連続メンション(everyoneも含む)、サーバー招待荒らし、クラ活を妨害するような行為

3. 不適切なツイート・リプライ

道徳の授業ちゃんとやってればわかると思います。

4. 荒らし行為(2回目は除隊)

厳重注意の2番とほぼ同じですが、2回目は除隊となります。
